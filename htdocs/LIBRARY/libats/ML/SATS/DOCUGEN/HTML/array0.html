ATSLIB/libats/ML/array0

The functions declared in this package are primarily for supporting ML-style processing of persisten arrays. Note that a plain array (as is in C) is not available in languages such as Java and ML due to the need for performing array-bounds checking at run-time. Instead, arrays in these languages are always bundled together with their sizes. In ATS, the type array0(T) is for an array paired with its size in which elements of the type T are stored.
