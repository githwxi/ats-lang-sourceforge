ATSLIB/libc/unistd

The functions declared in this package are mostly the API for the corresponding libc-functions declared in [unistd.h], and many more details on these functions can often be found in the Linux Programmer's manual. See unistd.sats for the SATS file in ATSLIB where these functions are declared.
