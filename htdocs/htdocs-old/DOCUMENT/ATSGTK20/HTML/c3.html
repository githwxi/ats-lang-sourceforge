ATS/Cario Basics

ATS is a rich programming language equipped with a highly expressive type system for specifying and enforcing program variants. Both dependent and linear types are available in ATS to support practical programming. ATS/GTK is the API in ATS for GTK+, a comprehensive toolkit for building graphical user interfaces. There are already many on-line tutorials on using GTK (e.g., this one), and the reader may want to use them as references when studying the current one, which focuses on using types in ATS to facilitate safe and reliable programming with GTK.
