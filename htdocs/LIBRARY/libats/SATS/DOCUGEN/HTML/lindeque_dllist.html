ATSLIB/libats/lindeque_dllist

    deque_vtype

    Synopsis

    absvtype
deque_vtype (a:vt@ype+, n:int) = ptr

    Description

    Given a type T and an integer N, the abstract viewtype deque_vtype(K, T) is for a deque storing N elements of the type T. Note that deque_vtype is co-variant in its first argument.
