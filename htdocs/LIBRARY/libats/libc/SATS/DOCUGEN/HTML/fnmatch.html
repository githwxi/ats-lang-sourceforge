ATSLIB/libc/fnmatch

The functions declared in this package are mostly the API for the corresponding libc-functions declared in [fnmatch.h], and many more details on these functions can often be found in the Linux Programmer's manual. See fnmatch.sats for the SATS file in ATSLIB where these functions are declared.
